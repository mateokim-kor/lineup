# LINE UP ⚽ — 조기축구 라인업 메이커

참석자만 고르면 쿼터별 라인업을 자동으로 짜주는 단일 파일 웹앱(PWA).

## 폴더 구조 (이 구조 그대로 GitHub 레포에 올리면 됨)
```
index.html        # 앱 본체 (단일 파일)
manifest.json     # PWA 설정 (앱 이름·아이콘·테마)
sw.js             # 서비스워커 (오프라인 실행)
icons/
  ├ icon-192.png            # PWA 아이콘
  ├ icon-512.png            # PWA 아이콘
  ├ apple-touch-icon-180.png# 아이폰 홈화면 아이콘
  ├ icon-1024.png           # 스토어 등록용
  └ icon-source.svg         # 아이콘 원본(수정용)
```

## 올리는 법
1. 위 파일들을 GitHub Pages 레포 루트에 커밋
2. 폰 브라우저로 페이지 열고 "홈 화면에 추가" → 바로 앱처럼 사용
3. 스토어 등록은 https://www.pwabuilder.com 에 페이지 주소 입력 → 패키지 생성
   - 구글 플레이: 개발자 계정($25) + 회원 12명 14일 비공개 테스트 후 출시
   - 애플: 개발자 계정($99/년) + 맥 필요

## 참고
데이터는 각 기기 브라우저에 로컬 저장(localStorage) — 기기 간 동기화는 없음.
인터넷이 없어도(운동장에서도) 오프라인으로 동작함.
